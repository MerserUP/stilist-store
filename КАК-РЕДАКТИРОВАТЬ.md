# Как управлять страницей Stilist Store

Всё управление — в файле `index.html`, в блоке `⭐ НАСТРОЙКИ` в самом верху.
Открыть файл можно любым текстовым редактором (TextEdit, VS Code, Заметки).
Изменили → сохранили → обновили страницу в браузере. Дизайн ниже блока настроек трогать не нужно.

---

## Добавить / изменить кнопку

Кнопки лежат списком в `buttons: [ ... ]`. Одна кнопка выглядит так:

```js
{
  icon: "instagram",       // иконка, варианты ниже
  style: "pink",           // "pink" — яркая, "glass" — стеклянная
  url: "https://...",      // куда ведёт
  title: "Большая надпись",
  sub: "Маленькая надпись",  // можно "" — тогда её не будет
},
```

**Добавить кнопку:** скопируйте весь блок от `{` до `},` и вставьте в список там,
где кнопка должна стоять. Порядок в списке = порядок на странице.

**Удалить кнопку:** сотрите её блок целиком, от `{` до `},`.

**Иконки:** `instagram`, `telegram`, `catalog`, `phone`, `pin`, `link`, `heart`, `star`, `cart`

---

## Филиалы

Список `branches: [ ... ]`. Один филиал:

```js
{
  name: "TTZ filiali",
  address: "TTZ 1-jilmassiv",
  mapUrl: "https://...",                  // ссылка «поделиться» из 2ГИС или Google Maps
  phones: ["+998 90 000-00-00"],          // номеров можно несколько, через запятую
},
```

Добавить третий филиал — так же копированием блока.

---

## Летающие товары

В `productPhotos: []` впишите пути к фото товаров без фона (PNG):

```js
productPhotos: ["img/gel-lak.png", "img/apparat.png"],
```

Файлы положите в папку `img` рядом с `index.html`.
Пока список пуст — летают рисованные флаконы.

---

## Режим работы

Строка `workingHours`. Время в формате `09:00 — 20:00` подсветится розовым само.

---

## Частые ошибки

- Пропала страница / всё сломалось → скорее всего потеряна запятая или скобка.
  Каждый блок `{ ... }` должен заканчиваться запятой.
- Текст с кавычкой внутри → пишите её как `\"`.
- После правок всегда обновляйте страницу (Cmd+R / потянуть вниз на телефоне).

Если страница уже опубликована в интернете, после правок её нужно опубликовать заново —
попросите Claude: «обнови опубликованную страницу».
